## Initial release

OffFILE v0.9.0 — Privacy-first, browser-based file processing toolkit.  
All operations run 100% client-side. No uploads, no servers, no tracking.

### Tools shipped

**PDF (14 tools)**

| Tool | Description |
|------|-------------|
| Merge PDF | Combine multiple PDFs into one |
| Split PDF | Extract selected pages into a new PDF |
| Extract Pages | Export specific pages as a ZIP |
| Remove Pages | Delete selected pages from a PDF |
| Rotate PDF | Rotate all or selected pages |
| Watermark PDF | Stamp text watermarks on every page |
| Page Numbers | Add footer/header page numbers |
| Compress PDF | Re-render pages at lower quality to reduce file size |
| PDF Metadata | Edit title, author, subject, creator fields |
| Reorder Pages | Drag-and-drop page reordering |
| Add Blank Page | Insert blank pages at end or after each page |
| Resize Page | Change page dimensions to A4, Letter, A3, or custom |
| PDF → Image | Render pages as PNG/JPEG |
| PDF → Text | Extract raw text content |

**Image (16 tools)**

| Tool | Description |
|------|-------------|
| Convert Format | Convert between PNG, JPEG, WebP |
| Compress | Reduce file size at configurable quality |
| Resize | Scale to specific dimensions |
| Rotate | Rotate 90°/180°/270° |
| Flip | Mirror horizontally or vertically |
| Watermark | Overlay text watermarks |
| Filters | Blur, sharpen, brightness, contrast, saturation |
| Strip Metadata | Remove EXIF/IPTC data |
| Grayscale | Convert to black and white |
| Add Background | Fill transparent areas with a solid color |
| Merge Images | Join images side-by-side or top-to-bottom |
| Crop | Crop with slider controls and canvas preview |
| Collage | Arrange multiple images in a grid |
| Color Palette | Extract dominant colors via median-cut |
| Image → PDF | Embed images into a PDF document |
| Image → Base64 | Encode as data URL, raw base64, or CSS |

### Architecture

- TypeScript + Vite, zero framework, zero server
- Hash-based SPA router (`#/`, `#/tool/:id`)
- All processing in a single Web Worker (`offfile.worker.ts`)
- PWA with offline caching via Workbox
- OffscreenCanvasFactory for pdfjs-dist Web Worker compatibility
